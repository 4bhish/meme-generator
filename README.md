# Meme Generator
A interactive web app where user can generate thier own memes .
## How to use:
- Enter your text of genric memes using top and bottom text inputs.
- click on get me to get meme template.
- you will have your meme now 
### Tech Used:
- imgFlip  API
- ReactJs
- CSS5
